Judul Proyek"Cannon Defense: Manual 2D Graphics Engine & Pixel-Based Transformation"(Sebuah simulasi game pertahanan berbasis koordinat piksel yang mengutamakan pengolahan grafis manual tanpa fungsi instan dari library).b. Konsep Grafika yang DigunakanRasterisasi Dasar: Proses mengubah bentuk geometris matematis menjadi kumpulan piksel pada layar menggunakan fungsi put_pixel.Sistem Koordinat 2D: Pengaturan posisi objek menggunakan sistem koordinat Kartesius $(x, y)$ pada kanvas digital.Transformasi Geometris 2D:Translasi: Perpindahan posisi meriam dan peluru.Rotasi: Perubahan sudut hadap meriam terhadap titik pusatnya.Skala: Penyesuaian ukuran aset gambar agar sesuai dengan resolusi layar.Refleksi: Pembalikan arah gerak objek saat menyentuh batas layar (efek pantul).Deteksi Tabrakan (Collision Detection): Menggunakan logika jarak Euclidean antara dua titik pusat objek.c. Algoritma yang DipakaiAlgoritma Garis Bresenham:Digunakan untuk menggambar garis lurus (peluru, batas deadline, dan sisi poligon). Algoritma ini dipilih karena sangat efisien (hanya menggunakan operasi bilangan bulat) dalam menentukan piksel terdekat dengan jalur garis asli.Algoritma Lingkaran Midpoint:Digunakan untuk menggambar target musuh berwarna merah. Algoritma ini menghitung posisi piksel pada satu oktan dan merefleksikannya ke tujuh oktan lainnya menggunakan simetri delapan titik.Algoritma Gambar Poligon (Wireframe):Digunakan untuk membentuk target kotak hijau. Algoritma ini menentukan empat titik sudut (vertex) dan menghubungkannya secara berurutan menggunakan fungsi garis hingga membentuk bidang tertutup.Algoritma Rotasi Piksel Manual:Untuk memutar meriam, setiap piksel pada gambar asli dipetakan ulang menggunakan matriks rotasi:$$x' = x \cos(\theta) - y \sin(\theta)$$$$y' = x \sin(\theta) + y \cos(\theta)$$d. Cara Menjalankan ProgramPersiapan Lingkungan:Pastikan Python sudah terinstal di komputer.Instal library pendukung (untuk membaca data piksel gambar) melalui terminal:pip install PillowPersiapan Aset:Pastikan file gambar bernama cannon.png berada dalam satu folder yang sama dengan file kode Python.Eksekusi:Jalankan program dengan perintah: python nama_file.py.Kontrol Permainan:Panah Kiri/Kanan: Menggeser posisi meriam (Translasi).Tombol A / D: Mengatur sudut kemiringan tembakan (Rotasi).Spacebar: Menembakkan peluru.Tombol R: Mengulang permainan (Restart) jika Game Over.Tombol Q: Keluar dari aplikasi (Quit).
